# Nino Maruszewski's dotfiles

My dotfiles!

## Instructions
1. Get git repo
    ```sh
    $ git clone git@github.com:NinoMaruszewski/.dotfiles.git ~/.dotfiles
    ```
2. Move to repo
    ```sh
    $ cd ~/.dotfiles
    ```
3. Make dotfiles on computer (add `--force` as needed)
    ```sh
    $ ./install
    ```
